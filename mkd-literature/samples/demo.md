## Extracts

> A plain block quote works for quoted prose or inset material.

```verse
The lamps were lit before the rain,
and every room recalled her name.
```

```extract
The notebook page smelled faintly of cedar and smoke.
```

```epigraph
Every return is a second invention.
-- Field notebook, winter edition
```

```letter
Dear Ada,

The harbor is louder now than when we left it.
```

```marginalia
Margin note: check whether the bell returns in chapter three.
```

## Structure

- Unordered list item
- [ ] Editorial to-do
- [x] Cleared note

1. Numbered observation
2. Numbered observation

Motif
